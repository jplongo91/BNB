````

## GIT / GITHUB

Initialiser un dépôt :
```
git init
```
Relier au dépôt distant (GitHub) :
```
git remote add origin https://lien_vers_le_dépôt_GitHub"
```
Ajouter les fichiers :
```
git add *
```
Nommer le commit :
```
git commit -m "le_nom_du_commit_ici"
```
Envoyer les fichiers :
```
git push origin master 
````
